# Password Generator

A simple and responsive Password Generator built using React.  
This project allows users to generate secure random passwords based on selected options like password length, numbers, and symbols.

---

## Features

- Generate random passwords
- Adjustable password length using slider
- Include Numbers option
- Include Symbols option
- Responsive UI
- Built with React functional components and Hooks

---

## Technologies Used

- React.js
- JavaScript
- CSS / Tailwind CSS
- Vite

---

```bash
├── main.jsx

```

---

## How It Works

1. User selects password length using the slider.
2. User can choose:
   - Include Numbers
   - Include Symbols
3. Clicking the **Generate** button creates a random password based on selected rules.
4. The generated password is displayed on the screen.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/password-generator.git
```

Go to the project folder:

```bash
cd password-generator
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
